# Fixtures

Fixture images are generated locally for the MVP verification run:

- `fixtures/images/sample.png`
- `examples/portfolio/assets/images/card.png`
- `examples/portfolio/assets/images/nested/card.png`
- `examples/dogfood/assets/images/social/open-graph.png`
- `examples/dogfood/assets/images/logos/devimg-mark.png`
- `examples/dogfood/assets/images/screenshots/cli-output.png`

They are deterministic PNG files generated from simple color gradients. Tests also create temporary corrupt and mismatched-extension files in isolated temp directories.
