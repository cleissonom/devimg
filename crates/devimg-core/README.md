# devimg-core

Core library for DevImg, a deterministic image pipeline for frontend repositories.

This crate owns config parsing, source scanning, image planning and transforms, manifest/report generation, budget checks, diagnostics, manifest export, compare output, and static review artifact rendering.

Most users should install the `devimg` CLI crate instead:

```bash
cargo install devimg
```

Source installs require Rust 1.88 or newer.

Repository: <https://github.com/cleissonom/devimg>
