# Documentation

This directory contains public DevImg documentation for users, contributors, and AI coding agents.

## User Guides

- `config.md`: complete `devimg.toml` reference.
- `action.md`: GitHub Action usage and inputs.
- `compatibility.md`: compatibility notes for the original `imgconvert` and `imgcrop` ideas.
- `public-distribution.md`: public install and distribution channels.
- `release.md`: maintainer release procedure.

## Project Internals

- `architecture.md`: core pipeline architecture and command semantics.
- `agent-contract.md`: ownership rules for AI coding agents working in DevImg-enabled projects.
- `agents.md`: practical agent workflow and generated instruction guidance.

## Roadmap

- `roadmap.md`: active forward plan after the completed v0.2 AI-assisted workflow releases.

## Dependency Compatibility

- `compatibility.md`: includes legacy tool behavior notes and the `image 0.25` migration compatibility notes.
